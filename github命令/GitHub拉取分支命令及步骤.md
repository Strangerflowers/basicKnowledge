# 2、git 获取不到gitLab创建的新分支
## 找到项目的路径并打开Git Bash Here然后去执行如下一条命令即可
## git fetch     刷新分支
## git branch -a 查看分支
## 这时候编辑器切换分支就可以了，小乌龟里也可以找到要切换的新分支。

# 3、创建分支
## git branch  分支名
## git branch  (查看分支)
## git checkout 分支名  （修改切换）
## git push  origin 分支名 
## git merge  分支名     （合并到当前分支，这步需要先切换到master）
